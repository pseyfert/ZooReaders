Detailed Description

    provide a class definition for logging output

a logstream can be used in much the same way as standard C++ output streams; however, one can define different logging facilities and severities, and one can select how severe an event being logged must be to actually show up in the output. this way, one can separate debugging output from more severe warnings and turn off debugging output during normal program runs. see below for examples of how to use/define a logstream.
