Detailed Description

Class that holds informations which were added to a particle in DaVinci.

flexible methods of adding float type information to a zoontuple enables you to put infos on the tuple which the maintainer wouldn't allow you to implement for public release. reads info(int ,double) method of LHCb::Particle
